# Madar

Landing page and waitlist for **Madar** — an AI automation service that handles repetitive business work, from customer replies to invoicing and inventory tracking.

🔗 **Live site:** https://aroze915-cmyk.github.io/madar/

## The idea

Every day, businesses lose hours to tasks that repeat themselves. Madar builds AI systems that connect directly to the tools you already use, so that work happens automatically instead of manually.

## Files

- `index.html` — the main landing page (waitlist form included)

## Tech

- Plain HTML/CSS/JS, no framework
- Waitlist submissions are sent via [Formspree](https://formspree.io)

🚧 In progress — waitlist is open now, first cohort launching soon.
