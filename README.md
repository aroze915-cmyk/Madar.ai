# مدار (Madar)

صفحة تعريفية وقائمة انتظار لمشروع **مدار** — خدمة أتمتة بالذكاء الاصطناعي لأصحاب الأعمال في العراق، تساعدهم يوفرون الوقت بالمهام المتكررة مثل الرد على الزبائن، الفواتير، ومتابعة المخزون.

🔗 **الموقع المباشر:** https://aroze915-cmyk.github.io/madar/

## الفكرة

كل يوم يضيع أصحاب الأعمال ساعات على مهام تنعاد نفسها. مدار يبني أنظمة أتمتة ذكية مربوطة مباشرة بأدوات العمل الحالية، بدون تعقيد أو حاجة لخبرة تقنية من صاحب العمل.

## الملفات

- `index.html` — الصفحة الرئيسية (تصميم كامل RTL بالعربي، وفيها فورم قائمة الانتظار)

## التقنية

- HTML/CSS/JS خالص، بدون أي إطار عمل (framework)
- التسجيل بقائمة الانتظار يتخزن مباشرة داخل الصفحة (مافي حاجة لسيرفر خارجي)

## الحالة

🚧 قيد التطوير — قائمة الانتظار مفتوحة الحين، والإطلاق قريباً في بغداد.
